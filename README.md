# ddolsoon-maps-platform
똘순이 Maps Platform Open API

### Routing Search API 
(도보/자동차/자전거 길찾기 API)

### GeoCoding API
우리나라 도로명/지번 주소를 지리적좌표(위경도 좌표 epsg:4326)로 변환해주는 API

### Reverse GeoCoding API
지리적좌표(위경도 좌표 epsg:4326)를 우리나라 도로명/지번 주소로 변환해주는 API

### 좌표 변환 API
제공가능할지는 아직 미지수 (라이브러리 조사중)


### Tech Stack
- osrm backend
- elasticsearch 7.10
- nginx
